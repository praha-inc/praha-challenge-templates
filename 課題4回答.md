## 課題4-1
内容は問いませんので、引数を受け取り、何らかの値を返却する関数を3つ作成してください。
### 回答
```ts

// 1. 簡単な関数: 数値の2乗を計算する
export function square(num: number): number {
  return num * num;
}

// 2. 簡単な関数: 文字列を逆順にする
export function reverseString(str: string): string {
  return str.split('').reverse().join('');
}

// 3. 複雑な関数: ユーザー情報を取得し、年齢に応じて処理を行う
interface User {
  id: number;
  name: string;
  age: number;
}

export async function processUserAge(userId: number): Promise<string> {
  try {
    const response = await axios.get(`https://api.example.com/users/${userId}`);
    const user: User = response.data;
    
    if (user.age < 18) {
      return `${user.name}さんは未成年です。`;
    } else if (user.age >= 65) {
      return `${user.name}さんはシニア割引の対象です。`;
    } else {
      return `${user.name}さんは成人です。`;
    }
  } catch (error) {
    throw new Error('ユーザー情報の取得に失敗しました');
  }
}

```

## 課題4-2
jestに関するクイズを作成してください。
jestそのものに関するクイズでも、jestを使ったテストケースに関するクイズでも構いません。
###
- 次のJestのマッチャーのうち、オブジェクトの深い等価性を確認するために使用されるのはどれ？
a. toBe()
b. toEqual()
c. toStrictEqual()
d. toMatch()
答え
b. toEqual()

- Jestでテストケースをグループ化するために使用される関数は次のうちどれですか？
a. group()
b. suite()
c. describe()
d. test.group()
答え
c. describe()

- 次のJestのライフサイクルフックのうち、すべてのテストが実行される前に一度だけ実行されるのはどれですか？
a. beforeEach()
b. afterEach()
c. beforeAll()
d. afterAll()
答え
c. beforeAll()
